Prerequisites

Before you begin, make sure you have the following:

Microsoft Account

Work or school account with Power Automate access

Microsoft Form

A form already created and ready to use

Planner Access

Access to Planner with at least one plan

Permissions

Necessary permissions to create flows

Step-by-Step Guide

Follow these 10 steps to create your automation flow

1

Access Power Automate

  • Navigate to Power Automate
  • Sign in with your Microsoft work or school account
  • Click on + Create in the left navigation pane
2

Create an Automated Cloud Flow

  1. In the Copilot prompt box, type:
when a new MS Forms response is submitted, create a task in Planner
  1. Click Next
  2. Review the suggested connections and click Create flow
  3. You'll land on the designer with a pre-configured flow
  1. Select Automated cloud flow from the Create menu
  2. Give your flow a name (e.g., "Forms to Planner Task")
  3. Search for "Microsoft Forms"
  4. Select the trigger When a new response is submitted
  5. Click Create
3

Configure the Microsoft Forms Trigger

  • In the When a new response is submitted card, click on the Form Id dropdown
  • Select the Microsoft Form you want to use from the list
  • This trigger will fire automatically every time someone submits your form
4

Add Get Response Details Action

  • Click + New step (or Add an action)
  • Search for "Microsoft Forms"
  • Select the action Get response details
  • In the Form Id field, select the same form you chose in Step 3
  • In the Response Id field, select Response Id from the dynamic content panel
5

Add Create Planner Task Action

  • Click + New step
  • Search for "Planner"
  • Select the action Create a task
  • Sign in to the Planner connector if prompted
6

Configure the Planner Task Details

Required Fields:

Group Id:

Select the Microsoft 365 Group/Team that contains your Planner

Plan Id:

Select the specific Planner plan where you want to create tasks

Title:

Use dynamic content from your form question responses

Optional Fields:

Bucket Id:

Choose which bucket/category to organize the task

Assignments:

Assign the task to specific users (use dynamic content if form collects assignee info)

Due Date Time:

Set a due date using dynamic content or expressions like: addDays(utcNow(), 7)

Description:

Combine multiple form answers to create detailed task descriptions

Priority:

Set task priority (1 = Urgent, 5 = Medium, 9 = Low)

7

Map Form Questions to Task Fields

Example mapping configuration:

Task Title
← "What task needs to be completed?"
Description
← "Task details" + "Additional notes"
Due Date
← "When is this due?"
Assigned to
← "Who should complete this?"

Tip: Click the lightning bolt icon to open the dynamic content panel and select form responses

8

Add Notification Action (Optional)

Notify someone when a task is created:

  • Click + New step
  • Search for "Send an email" (Office 365 Outlook)
  • Configure To, Subject, and Body using dynamic content
9

Save and Test Your Flow

  • Click Save at the top of the screen
  • Go to your Microsoft Form and submit a test response
  • Return to Power Automate and check the Run history
  • Verify the task was created in your Planner
10

Monitor and Troubleshoot

  • View flow run history by clicking on your flow name
  • Each run shows success/failure status with detailed logs

Common Issues:

  • • Incorrect Plan/Group IDs
  • • Missing required fields
  • • Permission issues

Advanced Tips

Take your automation to the next level

Conditional Logic

Create different tasks based on form responses

  • • Add Condition actions
  • • Check form answers (e.g., Priority = "High")
  • • Route to different task configurations

Multiple Assignments

Assign tasks to multiple people

  • • Use Parse JSON action
  • • Handle multiple user IDs
  • • Create Apply to each loops

Dynamic Buckets

Route tasks to different buckets automatically

  • • Add Switch or Condition
  • • Check form response category
  • • Use different Bucket IDs per category
